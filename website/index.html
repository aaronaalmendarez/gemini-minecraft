All Commands

Everything you need to know

/chat <text> Talk to the AI
/chat clear Clear conversation history
/chat cancel Cancel current request
/chathelp Show all commands
/chat config Interactive settings menu
/chatmodel flash|pro|auto Set AI model
/chatretries <0-10> Set retry limit
/chatsidebar on|off Toggle sidebar
/chatsounds on|off Toggle sounds
/chatparticles on|off|minimal Toggle particles
/chat history Show last 5 exchanges
/chat history all Show full history
/chat export Export to TXT
/chat export all json Export all as JSON
/chatkey <key> Save your API key
/chatkey clear Remove your key
/chatkey default <key> Set server default
/chatkey info Show key sources
